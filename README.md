# Mahmoud Sobhy - Professional Portfolio

## Project Overview

Welcome to the professional portfolio of Mahmoud Sobhy, showcasing skills, projects, and experience in full-stack development. The portfolio features multiple sections including an introduction, detailed information about the developer, showcases of projects, and contact information, all designed to provide a comprehensive view of Mahmoud's capabilities and services.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd <project-directory>
   ```
3. Open the `index.html` file in your preferred web browser.

## Usage

Once the project is opened in a web browser, you can explore the following sections:

- **Home**: Introduction with a brief summary of Mahmoud Sobhy.
- **About**: Background, education, and technical skills.
- **Projects**: Display of featured projects highlighting technologies used.
- **Contact**: A form to get in touch, along with links to social profiles.

## Features

- Responsive design for optimal viewing across various devices.
- Sections for a professional biography, educational background, and technical skills.
- Showcase of projects with descriptions and technologies used.
- Contact form for easy outreach.
- Inclusion of social media links.

## Dependencies

The project relies on the following CSS frameworks and libraries:

- [Google Fonts](https://fonts.google.com/) for typography:
  - **Inter** and **Poppins** font families.
  
Since this is a static site with no JavaScript libraries specified in a package.json file, the dependencies are primarily related to stylesheets.
